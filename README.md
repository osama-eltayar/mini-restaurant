## Installation

```
$ git clone https://github.com/osama-eltayar/mini-restaurant.git
```

```
$ cd mini-restaurant
```

```
$ composer install
```

```
$ cp .env.example .env
```

```
$ php artisan key:generate
```
- add database credentials in .env file
```
$ php artisan artisan:migrate --seed
```
```
$ php artisan serve
```
a postMan collection attached to the project
